HTML Grundgerüst
In einem Grundgerüst muss erstmal festgelegt werden, dass die Webseite mit html dargestellt wird. Desshalb beginnt man das Dokument mit dem "html" Tag. Um eine Webseite richtig darzustellen benötigt man einen "head" und einen "body". In diesen können alle weiteren Inhalte eingefügt werden.
Standard Tags

Überschriften

Überschriften werden durch "h" und eine Zahl dargestellt (z.B. h1). Je kleiner die Zahl desto kleiner die Überschrift.

Text

Den normalen Text kann man einfach in den "body" schreiben.

Listen

Eine Liste kann man mit dem "ol" Tag oder dem "ul" Tag darstellen.

Tabellen

Tabellen stellt man mit dem "table" Tag dar. Mit dem "tr" Befehl legt man die Zeile fest und mit "td" erzeugt man eine Spalte

Listen

Der "li" Tag erzeugt eine Aufzählung. Jede einzellne Aufzählung muss neu mit diesem Tag versehen werden.
